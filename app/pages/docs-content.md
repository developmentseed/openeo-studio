Welcome to the Open EO Editor documentation page. Here you will find all the necessary information to get started with the Open EO Editor.

This editor is designed to facilitate the creation and management of Open EO processes, providing a user-friendly interface for users to interact with [Open EO services](https://openeo.org/).

## Markdown Examples

Here are some examples of what you can do with markdown:

### Text Formatting
- **Bold text** using `**text**`
- *Italic text* using `*text*`
- ~~Strikethrough~~ using `~~text~~`

### Code
Inline code uses `backticks`, while code blocks use triple backticks:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
